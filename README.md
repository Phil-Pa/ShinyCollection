SolidSabisShinySammlung ist eine App, die für sogenannte Shiny-Hunter gedacht ist.
Shiny-Hunter benutzen ausgewählte Methoden, um Shiny Pokemon zu finden. Die Shiny Pokemon können sich auf
verschiedenen Spielen befinden, oder der Spieler hat sie alle auf ein Spiel übertragen.

In beiden Fällen ist es schwer, einen Überblick über die eigene Shiny Sammlung zu behalten.
SolidSabisShinySammlung ermöglicht diesen Überblick und gleichzeitig die Verwaltung der eigenen Sammlung.
