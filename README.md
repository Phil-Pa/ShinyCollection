## Was ist SolidSabisShinySammlung?

SolidSabisShinySammlung ist eine App, die für sogenannte Shiny-Hunter gedacht ist.
Der Name ist von Freunden inspiriert, die diese App benutzen.
Shiny-Hunter benutzen ausgewählte Methoden, um Shiny Pokemon zu finden. Die Shiny Pokemon können sich auf
verschiedenen Spielen befinden, oder der Spieler hat sie alle auf ein Spiel übertragen.

In beiden Fällen ist es schwer, einen Überblick über die eigene Shiny Sammlung zu behalten.
SolidSabisShinySammlung ermöglicht diesen Überblick und gleichzeitig die Verwaltung der eigenen Sammlung.

## Bedienung und Funktionen

Es gibt mehrere Tabs. Der erste ist die Shinyliste, hier befinden sich die bereits gefangenen Shiny Pokemon.
Man kann in den anderen Tabs die Shiny Pokemon hinzufügen, die man plant zu bekommen und kann diese dann in die
Shinyliste verschieben. Durch kurzen Berühren kann man die Encounter (Begegnungen) eines Pokemon erhöhen, bis man das Shiny gefunden hat.

Dann kann man durch langes Berühren ein Menü öffnen, in dem man:
- einen versehentlich erhöhten Encounter um 1 verringern kann
- das Pokemon löschen kann
- das Pokemon in die Shinyliste verschieben kann

Eine wichtige Funktion ist das Importieren und Exportieren der Daten, falls man ein neues Smartphone kauft
oder für die Datensicherung. Durch "Daten exportieren" werden die Daten als Text in die Zwischenablage kopiert
und können vom Benutzer an einen Ort seiner Wahl gespeichert werden. Für den Import müssen sich die zu importierenden
Daten ebenfalls wieder in der Zwischenablage befinden.
